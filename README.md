# KindleNotes2Txt
1. 下载仓库   
    `git clone https://github.com/weizhixiaoyi/KindleNotes2Txt.git`

2. kindle连接电脑，将My Clippings.txt文件拷贝至KindleNotes2Txt文件夹。  

3. 运行kindle_notes.py文件  

   `python kindle_notes.py`
